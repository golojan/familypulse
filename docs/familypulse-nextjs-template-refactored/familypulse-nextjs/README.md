# FamilyPulse Next.js Landing Blog Template

A mobile-first FamilyPulse blog/podcast homepage built with Next.js App Router, Tailwind CSS v4, and reusable React components.

## Structure

```txt
app/
  globals.css
  layout.tsx
  page.tsx
components/
  article-card.tsx
  audio-player.tsx
  featured-articles.tsx
  hero-slider-card.tsx
  logo.tsx
  media-grid.tsx
  mobile-bottom-nav.tsx
  news-card.tsx
  newsletter-card.tsx
  podcast-cta.tsx
  podcast-episode-list.tsx
  podcast-highlights.tsx
  quick-topics.tsx
  section-header.tsx
  site-header.tsx
  trending-topics.tsx
  video-card.tsx
lib/
  familypulse-data.ts
```

## Run

```bash
npm install
npm run dev
```
